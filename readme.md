# Sales Performance Dashboard (Tableau)

## Project Overview

This project analyzes retail sales data and presents insights using an interactive Tableau dashboard.

The goal of the dashboard is to identify patterns in sales performance across regions, product categories, and time.

---

## Key Insights

The dashboard answers the following business questions:

* Which region generates the highest sales?
* Which product category is most profitable?
* How do sales change over time?
* Which cities contribute the most revenue?

---

## Dashboard Visualizations

The dashboard includes:

* Sales by Region (Bar Chart)
* Profit by Category (Bar Chart)
* Monthly Sales Trend (Line Chart)
* Top Cities by Sales (Bar Chart)

Interactive filters allow users to explore the data by region.
